Локальный запуск
- Рекомендуется статический сервер с fallback на index.html.
- Пример: `npx serve -s .` и открыть выданный адрес.

Структура
- /core — дизайн-система и компоненты
- /shared — общая логика, API, роутер
- /pages — страницы
